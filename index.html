1. Information We Collect

Vinly does not collect, store, transmit, or process personal information from users.

The App:

  • does not require user registration
  • does not request personal data
  • does not track user activity
  • does not collect location information
  • does not access contacts, camera, or microphone

2. Device and Technical Information

While the application itself does not collect personal data, certain technical information may be processed automatically by the operating system or application distribution platforms, such as Google Play, to ensure proper functionality.

This may include:

  • device model
  • operating system version
  • application version
  • basic network information
This information is handled by platform providers and is not accessible to the developer of Vinly.

3. Permissions

Vinly does not request access to sensitive device permissions, including but not limited to:

  • Location
  • Camera
  • Microphone
  • Contacts
  • Storage
  • Phone state

4. Third-Party Services

Vinly does not integrate third-party SDKs or services that collect user data, such as:

  • analytics services
  • advertising networks
  • tracking tools
  • external user profiling services

5. Data Sharing

Because Vinly does not collect personal data, no user information is shared with third parties.

6. Data Retention

Vinly does not store any personal data. As a result, there is no user data retained by the application.

7. Children’s Privacy

Vinly is suitable for users of all ages.

Since the application does not collect personal information, it complies with privacy protection requirements related to children.

8. Security

Although the application does not collect user data, we take reasonable steps to ensure that the application itself is developed following standard security practices.

9. Changes to This Privacy Policy

We may update this Privacy Policy from time to time.

Any updates will be published on this page with an updated Last updated date.
